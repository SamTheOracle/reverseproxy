In order to use certificates, if csr is autogenerated by website, the private.key must be used to generate keystore.

1) Generate pkcs12 keystore with certificate and private.key
 openssl pkcs12 -export -in certificate.crt -inkey private.key -certfile certificate.crt -out keystore.p12

2) using keytool, import keystore.p12 in a new keystore using store type pkcs, because it is a public type, while jks uses a proprietary format (not trusted)
  keytool -importkeystore -srckeystore keystore.jks -destkeystore keystore.jks -deststoretype pkcs12
